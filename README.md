### Embed Message help
For the `embed` command, please add at least 7 lines to your message, as follows.

```=embed Title of embed
Title URL
Description
Author
Author URL
Author Image Link
Footer
```

You might want to add fields to your embed. For every field, insert 3 lines to the bottom of your command.
```Field Title
Field Description
Field Inline
```
